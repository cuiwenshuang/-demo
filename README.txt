【使用步骤】
1.用数据线连接安卓手机到电脑
2.启动答题外挂v1.0
3.在答题直播过程中 题目出现时 按任意键加回车 外挂会自动识别题目并启动浏览器自动搜索

【注意事项】
1.1.0版本只支持安卓机4.0以上版本
2.测试手机屏幕过大、分辨率过高等问题可能导致识别失败（测试机为小米5,识别无问题，尽量选择与小米5屏幕大小、分辨率差不多的手机）
3.出现adb error等字样 需检查 
  1>手机是否开启USB调试模式
  2>是否有手机助手、豌豆荚等软件占用adb 5053端口 在任务管理器中关闭进程 目前已知错误进程为：
    tadb(腾讯、金山) sjk(未知)等。。。 
4.确保手机不黑屏
5.这个包一定要放到D盘 再解压缩跑
6.浏览器的窗口不用可以关掉 但是浏览器别关
